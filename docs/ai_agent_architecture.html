🛡️ Agent 1 — SENTINEL Signal Filter & Noise Removal

Monitors Discord channel 1507850557505671209, parses TrendVision alerts, rejects noise, builds dynamic watchlist

🔍
Signal Parser
Real-Time Alert Ingestion
Regex engine extracts structured data from TrendVision's raw alert format. Every field is parsed and normalized for downstream analysis.
GLXG • #3 · BREAKOUT · ↑58% · $2.05 • FT 1.7M · MC 2.0M · RV 65x · 1V 179K

Parsed → { symbol: "GLXG", rank: 3, signal: "BREAKOUT", change: +58%, price: 2.05, float: 1.7M, mktcap: 2.0M, relVol: 65x, vol1m: 179K }
  • Regex parser for TrendVision format: SYMBOL · #RANK · TYPE · ↑%CHG · $PRICE • FT · MC · RV · 1V
  • Whale alert parser: 🐋 SYMBOL - LARGE WHALE · Direction · Price · Shares · Value
  • Borrow/short data parser: SYMBOL · BORROW · CTB% | SI%
  • NEWS attachment detection and link extraction
  • Deduplication — same ticker within configurable cooldown window

Dynamic Watchlist
Auto-Build & Auto-Expire

📊 Agent 2 — ANALYST Technical Analysis & Setup Detection

Continuously monitors the dynamic watchlist for high-probability trade setups using VWAP, Fibonacci, and momentum patterns

📐
VWAP Engine
Penny Stock VWAP Analysis
  • VWAP Reclaim: Price crosses above VWAP after dip → bullish setup
  • VWAP Hold: Price tests VWAP as support and bounces → entry signal
  • VWAP Distance: Price >20% above VWAP → overextended, wait for pullback
  • Volume at VWAP: High volume at VWAP test = strong support confirmation
  • Real-time VWAP calculated via UPH streaming price data
🔢
Fibonacci Analyzer
Pullback & Extension Levels
  • Fib Retracement: 38.2%, 50%, 61.8% pullback from day's high → dip buy zones
  • Fib Extension: 161.8%, 261.8% targets for profit-taking levels
  • Confluence: Fib level + VWAP overlap = high-probability zone
  • Dynamic Recalc: Auto-recalculates when new high/low is set
  • Intraday low-to-high range for penny stock scalping
⚡
Momentum Scanner
Pattern & Volume Analysis
  • Volume Surge: 1-min volume spike > 3x average → breakout signal
  • Consolidation Break: Tight range followed by expansion → entry
  • Higher Lows: Each pullback holds above prior low → trend strength
  • Relative Volume Decay: RV dropping alert-over-alert → momentum fade
  • Whale Correlation: Large whale buy during consolidation → smart money signal
🎯

⚡ Agent 3 — EXECUTOR Robinhood MCP Trade Execution

Executes trades on Robinhood via official MCP endpoint with risk management integration

🤖
MCP Execution Engine
Robinhood Official API
Connects to agent.robinhood.com/mcp/trading via OAuth 2.1. All orders go through the official agentic trading channel — no unofficial API risk.
  • Order Types: Market, Limit, Extended Hours
  • Position Sizing: % of agentic account buying power (configurable per score tier)
  • Entry Strategy: Limit order at VWAP/Fib level from Agent 2 analysis
  • Slippage Guard: Reject if current price > X% above Agent 2's target entry
  • Max Positions: Configurable concurrent position limit (default 3)
  • Daily Trade Limit: Integrates with existing Daily P&L Limit service
🛡️
Risk Integration
Existing Risk Engine (position_monitor.py)
After Agent 3 places a trade, the existing BotifyTrades risk engine takes over position management — no new code needed for exits.
  • Stop Loss: Set at Fib/VWAP support level from Agent 2 (typically -8% to -15%)
  • Profit Targets: Tiered exits at Fib extensions (30% / 50% / 100% trim schedule)
  • Trailing Stop: Activates after PT1 hit — locks in gains on momentum runners
  • Leave Runner: Keep 10-25% of position for multi-day holds on low-float runners
  • Circuit Breaker: Existing daily loss limit stops all agents if hit
  • Software SL: Risk engine evaluates every ~1s — critical for penny stock speed
