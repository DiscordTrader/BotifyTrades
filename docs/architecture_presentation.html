The synchronization service reconciles positions, detects fills, and resolves state drift between the platform and broker accounts.

Speaker Notes
Security is architected at every layer. Broker credentials are encrypted using Windows DPAPI, binding tokens to the user's OS account. Brokers that support OAuth 2.0 (Schwab, Tastytrade, Questrade, Zerodha, Upstox) use standard authorization code flows with automatic token refresh. All broker API communication uses TLS via the certifi CA bundle. The platform logs every trade decision, signal processed, and risk trigger with timestamps for signal-to-fill traceability.
