# QuantumPulse UI Mockups

This folder contains 8 high-fidelity visual mockups that showcase the QuantumPulse trading platform. These mockups are designed to be used in the landing page, marketing materials, pitch decks, and sales presentations.

## Mockup Files

### 1. Discord Signal Monitor (`1-discord-signals.html`)
- **Purpose**: Shows live Discord channel monitoring with automated signal parsing
- **Key Features**: 24/7 monitoring badge, multiple channel signals, BTO/STC tags, timestamps
- **Premium Indicator**: AI pre-trade validation banner (upgrade CTA)

### 2. Automated Trade Execution (`2-trade-execution.html`)
- **Purpose**: Demonstrates multi-broker trade execution across Webull, Alpaca, and IBKR
- **Key Features**: Real-time execution status, slippage protection, fill rates, execution speed
- **Premium Indicator**: AI-based pre-trade validation banner

### 3. Live Trading Positions (`3-live-positions.html`)
- **Purpose**: Real-time position tracking with P/L monitoring
- **Key Features**: Risk level indicators (Low/Medium/High), win rate, avg return, hide balance toggle
- **Premium Indicator**: Real-time AI risk analysis badge and banner

### 4. Options Chain Preview (`4-options-chain.html`)
- **Purpose**: Live options chain with Greeks and market data
- **Key Features**: ATM strike highlighting, IV tracking, Delta, probability calculations
- **Premium Indicator**: AI strike selection and Live Greeks banners

### 5. Performance Leaderboard (`5-leaderboard.html`)
- **Purpose**: User and channel performance rankings
- **Key Features**: Win rates, P/L percentages, medal rankings (gold/silver/bronze)
- **Premium Indicator**: Historical analytics banner (Pro feature)

### 6. AI Trade Analysis (`6-ai-analysis.html`)
- **Purpose**: GPT-powered trade analysis with technical indicators
- **Key Features**: Success probability gauge, technical indicators (RSI, MACD, etc.), risk/reward analysis
- **Premium Indicator**: Full blur overlay with lock icon and upgrade CTA (subscription required)

### 7. Risk Management Panel (`7-risk-management.html`)
- **Purpose**: Automated risk controls and position protection
- **Key Features**: Auto profit targets (20%), stop losses (10%), trailing stops (5%), real-time risk scoring
- **Premium Indicator**: Advanced risk AI engine banner (Pro/Elite only)

### 8. Full Dashboard Composite (`8-dashboard-composite.html`)
- **Purpose**: Complete dashboard view combining all panels
- **Key Features**: All components in one view - perfect for hero sections and presentations
- **Premium Indicator**: AI analysis section shown with blur/lock overlay

## Shared Styling

All mockups use:
- **CSS Framework**: `mockup-styles.css` (shared across all mockups)
- **Color Scheme**: Dark trading theme (#0a0e17, #141821, #1a1f2e)
- **Accent Colors**: Teal (#00d4ff), Blue (#0080ff), Success (#00ff88), Warning (#ffd700), Error (#ff6b6b)
- **Typography**: Inter font family for clean, professional look
- **Consistent Components**: Data tables, status badges, risk indicators, premium banners

## Usage

### For Landing Page
The mockups are integrated into `landing-page/index.html` with embedded iframes in the "See QuantumPulse in Action" section.

### For Screenshots
Each mockup can be opened individually in a browser and screenshot for:
- Marketing materials
- Pitch decks
- Social media posts
- Documentation
- Sales presentations

### For Demos
Open `index.html` in this folder to view all mockups in a gallery format with descriptions.

## Premium vs Free Trial Features

The mockups clearly distinguish between:

**FREE TRIAL:**
- View dashboard and positions
- Monitor Discord signals
- See basic execution logs
- View leaderboards (limited data)

**PAID SUBSCRIPTION (Pro/Elite):**
- AI trade analysis (GPT-4 powered)
- AI pre-trade validation
- AI strike selection
- Real-time risk scoring
- Advanced Greeks tracking
- Historical performance analytics
- Full leaderboard data

All premium features are marked with:
- 🔒 Lock icons
- Blur overlays
- "Upgrade to Pro" CTAs
- Premium feature badges

## Design Philosophy

These mockups follow a professional trading platform aesthetic with:
- Clean, data-dense layouts
- Clear visual hierarchy
- Consistent color-coding (green for profit, red for loss, blue for info)
- Professional typography and spacing
- Realistic dummy data that looks authentic
- Enterprise-grade polish
