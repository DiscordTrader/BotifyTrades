================================================================================
🔧 PROTECTION TECHNOLOGIES
================================================================================

STANDARD BUILD:
  • PyInstaller (Single-file packaging)
  • AES-256 Bytecode Encryption
  • UPX Compression
  • Symbol Stripping (Linux)

HARDENED BUILD:
  • PyArmor BCC Mode (Python → C compilation)
  • PyArmor RFT Mode (Irreversible renaming)
  • String Encryption (All literals encrypted)
  • Import/Call Assertions (Runtime verification)
  • AES-256 Bytecode Encryption
  • UPX Ultra Compression
  • Symbol Stripping (Linux)

⚙️ REQUIREMENTS

ALL BUILDS:
  ✓ Python 3.8+
  ✓ pip install pyinstaller pycryptodome

HARDENED BUILDS ONLY:
  ✓ pip install pyarmor

OPTIONAL (RECOMMENDED):
  ✓ UPX compressor
    Windows: https://upx.github.io/
    Linux: sudo apt install upx-ucl

================================================================================
📊 BUILD WORKFLOW
================================================================================

STANDARD BUILD:
  1. Clean previous builds
  2. Generate random AES-256 key
  3. Package with PyInstaller
  4. Compress with UPX
  5. Create distribution package
  
  Time: ~2 minutes
  Result: Single-file executable

HARDENED BUILD:
  1. Clean previous builds
  2. Obfuscate code with PyArmor BCC+RFT
  3. Generate random AES-256 key
  4. Package obfuscated code with PyInstaller
  5. Compress with UPX ultra
  6. Create distribution package
  
  Time: ~5 minutes
  Result: Maximum-protection executable

================================================================================
🎯 USE CASES

USE STANDARD BUILD WHEN:

USE HARDENED BUILD WHEN:
  ✓ Commercial distribution

  ✓ Maximum security required
  ✓ Professional deployment

================================================================================
📝 NOTES
================================================================================

• Build on target platform (Windows exe on Windows, Linux binary on Linux)
• Each build generates unique encryption key
• PyArmor free tier: Limited to 10 registration uses
• UPX reduces file size by 30-50%
• Code signing not included (add separately for production)
• Cross-compilation not supported

🏁 GETTING STARTED
================================================================================

1. Install dependencies:
   pip install pyinstaller pycryptodome pyarmor

2. Choose your build:
   Windows: build\windows\build_hardened.bat
   Linux:   ./build/linux/build_hardened.sh

3. Wait ~5 minutes

4. Find executable in:
   build\windows\dist_hardened\  (Windows)
   build/linux/dist_hardened/    (Linux)

5. Distribute to customers!
