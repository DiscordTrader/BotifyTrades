📋 OPTION 1: RUN ON YOUR COMPUTER (Python)
═══════════════════════════════════════════════════════════════════════

STEP 1: Install Python
   ➜ Download Python 3.11+ from https://www.python.org/downloads/
   ➜ During install, CHECK "Add Python to PATH"

STEP 2: Download Bot
   ➜ Download this project as ZIP from Replit
   ➜ Extract to C:\TradingBot (Windows) or ~/TradingBot (Mac/Linux)

STEP 3: Install Dependencies
   ➜ Open Command Prompt/Terminal in bot folder
   ➜ Run: pip install -r requirements.txt
   ➜ Wait 2-3 minutes for installation

STEP 4: Configure
   ➜ Copy config.ini.example to config.ini
   ➜ Edit config.ini with your credentials:
      - Discord token (use GET_DISCORD_TOKEN.html)
      - Webull credentials (use GET_WEBULL_TOKENS.html)
      - API keys (Alpha Vantage, Finnhub, OpenAI)
      - License key

STEP 5: Run Bot
   ➜ Run: python src/selfbot_webull.py
   ➜ Open browser: http://127.0.0.1:5000
   ➜ Done! ✅


📦 OPTION 2: BUILD WINDOWS .EXE (Share with Anyone)
═══════════════════════════════════════════════════════════════════════

FOR BUILDERS (You):
   
   STEP 1: Open Command Prompt in bot folder
   
   STEP 2: Run build script
      ➜ build_exe.bat
      ➜ Wait 3-5 minutes
   
   STEP 3: Create distribution package
      ➜ create_distribution.bat
      ➜ Creates: TradingBot-Distribution.zip
   
   STEP 4: Share the ZIP file
      ➜ Upload to Google Drive, Dropbox, etc.
      ➜ Send download link to customers


FOR CUSTOMERS (Them):

   STEP 1: Extract ZIP
      ➜ Extract TradingBot-Distribution.zip to C:\TradingBot
   
   STEP 2: Run the EXE
      ➜ Double-click DiscordTradingBot.exe
      ➜ Follow interactive setup wizard
   
   STEP 3: Open Web GUI
      ➜ Browser opens automatically: http://127.0.0.1:5000
      ➜ Done! ✅
   
   📌 NO PYTHON NEEDED! Just extract and run!

🔧 QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Bot won't start?
   ✓ Check Python version: python --version (must be 3.11+)
   ✓ Reinstall: pip install -r requirements.txt --force-reinstall
   ✓ Check config.ini for typos

Web GUI not loading?
   ✓ Check if Flask started (terminal should show "Running on...")
   ✓ Try http://localhost:5000 instead
   ✓ Check firewall/antivirus

Build .exe fails?
   ✓ Use Python 3.11 (not 3.12+)
   ✓ Run: PYINSTALLER_FIX.bat
   ✓ Temporarily disable antivirus

License issues?
   ✓ Get machine ID: GET_MACHINE_ID.bat
   ✓ Check expiration date
   ✓ Verify no typos in key
