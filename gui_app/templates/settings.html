🪶 Robinhood (LIVE Only)

Connect your Robinhood account for stock and options trading

⚠️ LIVE TRADING ONLY - NO PAPER MODE

Robinhood does NOT have paper trading. All trades executed through this integration are REAL and use REAL MONEY. This uses an unofficial API that may break without notice.

Not connected

Your Robinhood account email

To get this code:
1. Robinhood App/Web → Settings → Security & Privacy
2. Enable Two-Factor Authentication → Select "Authenticator app"
3. Click "Can't scan?" or "Enter code manually"
4. Copy the 16-character alphanumeric code shown (e.g., KACUJH6EKZZCVZ6K)
If you don't provide this, you'll need to manually enter SMS codes during login.

📱 2FA REQUIREMENT

Robinhood only works with 2FA push notification (Authenticator app / TOTP). SMS-based 2FA is not supported for automated login. You must switch your Robinhood 2FA to an authenticator app and provide the TOTP secret above.

Options Note: Robinhood only supports LIMIT orders for options (no market orders). The bot will automatically use limit prices based on current bid/ask.

⚠️ API Stability: This uses an unofficial API (robin-stocks). Robinhood can break this at any time without notice. Check GitHub issues if authentication fails.

How to Extract Tokens from Browser
  1. Login to app.webull.com in Chrome (complete any captcha manually)
  2. Press F12 to open DevTools, go to Network tab
  3. Refresh the page (Ctrl+R)
  4. Click on any API request, look for Response or Headers
  5. Find and copy: access_token and did
  6. Paste them in the fields above
