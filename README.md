# BotifyTrades

**Automated Multi-Broker Trading Bot**

Execute Discord/Telegram trading signals across 8+ brokers with AI-powered signal parsing, risk management, and real-time position monitoring.

## Features

- 🤖 AI Signal Parsing — Auto-detect BTO/STC/conditional signals from 140+ formats
- 📊 8 Broker Support — Webull Official, Schwab, IBKR, Alpaca, Tastytrade, Robinhood, Webull Legacy, Trading212
- 🛡️ Risk Engine — SL/PT/trailing/dynamic SL/EMA/giveback guard per channel
- ⚡ Real-Time Monitoring — MQTT/WebSocket streaming with <50ms tick-to-eval
- 🧠 AI Co-Pilot — MCP server + natural language bot control
- 📱 Web Dashboard — Flask-based GUI for positions, trades, settings
