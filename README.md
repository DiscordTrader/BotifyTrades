# BotifyTrades

Automated multi-broker trading bot for Discord signal execution.

## Download

Get the latest release for your platform from the [Releases page](https://github.com/DiscordTrader/BotifyTrades/releases).

| Platform | Asset |
|----------|-------|
| Windows | `BotifyTrades-windows.exe` |
| Linux | `BotifyTrades-linux` |
| macOS (Intel) | `BotifyTrades-macos-intel.tar.gz` |
| macOS (Apple Silicon) | `BotifyTrades-macos-silicon.tar.gz` |

## Getting Started

1. Download the latest release for your OS
2. Run the executable — a setup wizard will guide you through broker configuration
3. Configure your Discord channels and risk settings via the web GUI
